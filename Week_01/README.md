``` C++
for (int i = 0; i < list.length - 1; i++)

```

* 左右边界 i, j, 向中间收敛

``` C++
for (int i =0, j = list.length - 1; i < j) {
    i++;
    j--;
}

```

### 4. 实战题目解析：3数之和、环形链表

#### 3数之和 （高频）
双指针法/夹逼法，做到滚瓜烂熟

经典写法
``` C++
while ( i < j && nums[ i ] == nums[ ++i ] );
while ( i < j && nums[ j ] == nums[ --j ] );
```
